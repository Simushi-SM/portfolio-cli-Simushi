HOW TO RUN THE CLI APP
When you open the CLI app it brings you a menu which has information categories about the ower.
Then you select a Number on the menu which has the Information which you want to find out more about.
You only select a number which appears on the menu,the rest of the numbers or letters are considered as invalid inputs.
